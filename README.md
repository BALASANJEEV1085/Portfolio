# Portfolio

> Personal developer portfolio built with React — showcasing projects, skills, and contact information.

## About

This is my portfolio website that highlights my projects, skills, and experience as a developer.  
It’s built with **React** and designed to be clean, responsive, and easy to navigate.

**Key highlights:**
- Professional overview of my work
- Project gallery with live previews
- Contact section integrated for easy reach

## Tech Stack

This project uses the following technologies:

- **React** — UI library
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Responsive design** for mobile and desktop
- **Background effect** — React Bits

## Installation & Setup

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/BALASANJEEV1085/Portfolio.git
2. Navigate into the project directory:
     ```bash
     cd Portfolio
3. Install dependencies:
     ```bash
     npm install
4. Start the development server:
     ```bash
     npm start
